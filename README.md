I'm a back-end software engineer (fullstack capable) based in Nigeria, passionate about building robust APIs, scalable server-side systems, and digital infrastructure that serves African and global markets. I've shipped production-grade software for a Nigerian government agency, helped grow an artisan booking platform to 80+ users with a 40% engagement boost, and built fintech and agri-tech tools that tackle everyday challenges across the continent. Beyond backend work, I'm on a mission to break into 2D game development — because great software should also be fun.
